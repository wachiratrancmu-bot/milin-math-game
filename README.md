# ⭐ ผจญภัยดาวคณิตของมิลิน

เกมฝึกทักษะคณิตศาสตร์ระดับ **ป.1** (เหมาะกับ ป.1–2) สร้างด้วย **React + Vite + TypeScript**
มีระบบ **สะสมดาวแลกรางวัล** และหน้า **ตั้งค่าสำหรับผู้ปกครอง** ปรับแต่งได้ทุกอย่าง

ออกแบบตามใบสอบเก็บคะแนนท้ายบทที่ 1 — ครบทั้ง 6 หัวข้อ:
จำนวนและค่าประจำหลัก · เปรียบเทียบจำนวน (>,<,=) · การกระจายจำนวน · การเรียงลำดับ · ส่วนรวม–ส่วนย่อย · การบอกอันดับ (+ แบบรูปของจำนวนเป็นโบนัส)

---

## ✨ ความสามารถหลัก

- **ข้อสอบ 2 รูปแบบ**: กากบาท (เลือกตอบ) และ เติมคำตอบ
- **คลังข้อสอบในตัว** กว่า 100 ข้อ 3 ระดับ (ง่าย/กลาง/ยาก) ครอบคลุมตั้งแต่นับจากภาพ เลขไทย ตัวหนังสือ จนถึงโจทย์สองขั้น เหมาะกับ ป.1–2
- **ระบบรางวัล 2 แบบพร้อมกัน**
  - สะสมดาวข้ามวัน (แต้มไม่หาย) — ตอบถูกครบตามเป้า (เริ่มต้น 30 ดวง) ได้ 🍦/🍬 1 ชิ้น
  - โบนัสดาวพิเศษเมื่อทำคะแนนเต็มในรอบเดียว
- **หน้าตั้งค่าผู้ปกครอง (ล็อกด้วย PIN)** ปรับได้:
  ชื่อลูก · ชื่อ/สัญลักษณ์รางวัล · เป้าดาวต่อรางวัล · โบนัสทำเต็ม · จำนวนข้อต่อรอบ ·
  โหมด (ฝึก/สอบ) · ระดับความท้าทาย · เลือกเปิด-ปิดหัวข้อ · คำใบ้ · เสียง ·
  **เพิ่มข้อสอบเองได้** · มอบรางวัล · ดูประวัติ · รีเซ็ต · เปลี่ยน PIN
- บันทึกความคืบหน้าอัตโนมัติในเครื่อง (localStorage) — ไม่ต้องเชื่อมต่ออินเทอร์เน็ต
- เล่นได้ทั้งคอมพิวเตอร์ แท็บเล็ต และมือถือ · มีเสียงและภาพเฉลิมฉลอง

> **PIN เริ่มต้นคือ `1234`** (เปลี่ยนได้ในหน้าตั้งค่าผู้ปกครอง)

## 🚀 วิธีรันบนเครื่อง (สำหรับพ่อแม่/นักพัฒนา)

ต้องมี [Node.js](https://nodejs.org) เวอร์ชัน 18 ขึ้นไป

```bash
npm install      # ติดตั้งครั้งแรกครั้งเดียว
npm run dev      # เปิดเล่นที่ http://localhost:5173
```

สร้างไฟล์สำหรับนำไปใช้จริง:

```bash
npm run build    # ได้ผลลัพธ์ในโฟลเดอร์ dist/
npm run preview  # ลองเปิดไฟล์ที่ build แล้ว
```

ตรวจคุณภาพโค้ด (CI จะรันให้อัตโนมัติก่อน deploy ทุกครั้ง):

```bash
npm run lint     # ตรวจด้วย ESLint
npm run test     # รัน unit test (Vitest)
npm run format   # จัดรูปแบบโค้ดด้วย Prettier
```

## 🌐 นำขึ้น GitHub Pages (ให้ลูกเล่นผ่านลิงก์ได้ทุกที่)

1. สร้าง repository บน GitHub แล้ว push โค้ดขึ้นไป (ดูด้านล่าง)
2. ไปที่ **Settings → Pages → Build and deployment → Source** เลือก **GitHub Actions**
3. ทุกครั้งที่ push ขึ้น branch `main` ระบบจะ build และเผยแพร่อัตโนมัติ
   (ตั้งค่าไว้แล้วในไฟล์ `.github/workflows/deploy.yml`)
4. รอสักครู่ เว็บจะอยู่ที่ `https://<ชื่อผู้ใช้>.github.io/<ชื่อ-repo>/`

push ครั้งแรก:

```bash
git init
git add .
git commit -m "เริ่มต้นเกมผจญภัยดาวคณิตของมิลิน"
git branch -M main
git remote add origin https://github.com/<ชื่อผู้ใช้>/<ชื่อ-repo>.git
git push -u origin main
```

> โปรเจกต์ตั้ง `base: './'` ไว้แล้ว จึงใช้ได้ทั้ง GitHub Pages และโฮสต์อื่น โดยไม่ต้องแก้ค่า

## 🛠 โครงสร้างโปรเจกต์

```
src/
  data/questions.ts        คลังข้อสอบ + ชื่อ/ไอคอนหัวข้อ + น้ำหนักตามใบสอบ
  lib/
    quiz.ts                สุ่มชุดข้อสอบ + ตรวจคำตอบ
    storage.ts             อ่าน/เขียน localStorage + ค่าเริ่มต้น
    sound.ts               เสียงเอฟเฟกต์ (Web Audio)
    confetti.ts            เอฟเฟกต์โปรยดาว
  state/
    SettingsContext.tsx    การตั้งค่าทั้งหมด (พ่อแม่ปรับได้)
    ProgressContext.tsx    ความคืบหน้า ดาว และระบบรางวัล
  components/
    Home.tsx               หน้าแรก + กล่องเก็บดาว
    Game.tsx               หน้าเล่น + ให้คะแนน
    QuestionView.tsx       แสดงโจทย์ (กากบาท/เติมคำ)
    ResultModal.tsx        สรุปผลรายหัวข้อ
    RewardModal.tsx        ฉลองเมื่อได้รางวัล
    ParentSettings.tsx     หน้าตั้งค่าผู้ปกครอง (ล็อก PIN)
    Visual.tsx             ภาพประกอบโจทย์
  App.tsx                  เชื่อมหน้าทั้งหมดเข้าด้วยกัน
legacy/                    เกมเวอร์ชันเดิม (ไฟล์ HTML เดียว) เก็บไว้อ้างอิง
```

## ➕ เพิ่มข้อสอบ

วิธีง่ายที่สุด: เปิด **ตั้งค่าผู้ปกครอง → เพิ่มข้อสอบเอง** กรอกในเกมได้เลย (เก็บในเครื่อง)

ถ้าต้องการเพิ่มถาวรในโค้ด: แก้ไฟล์ [`src/data/questions.ts`](src/data/questions.ts)
แล้วเพิ่มอ็อบเจกต์ตามรูปแบบที่มีอยู่ (กำหนด `topic`, `level`, `kind`, `text`, `answer` และ `choices` ถ้าเป็นแบบกากบาท)
